# Smart Study Companion

#### Video Demo: <https://youtu.be/CQByC-BuLx8>

#### Description:

Smart Study Companion is a Python application designed to help students study through quizzes, track their progress, and manage study sessions.

The program allows users to:
- Take quizzes from different subjects using the Open Trivia Database API.
- Choose the number of questions and difficulty level.
- Save quiz scores to a CSV file.
- View previous quiz scores.
- Generate a progress chart showing performance over time.
- Use a Pomodoro timer to manage study sessions.

## Files

### project.py
Contains the main program logic, including quiz generation, score saving, progress tracking, and the Pomodoro timer.

### test_project.py
Contains unit tests for selected functions such as percentage calculation and time formatting.

### score.csv
Stores quiz results including date, subject, difficulty, score, and total questions.

### Progress.png
Generated by the program when the user chooses to view their progress chart.

## Design Choices

I chose to use the Open Trivia Database API because it provides a large collection of quiz questions from different categories.

Quiz results are stored in a CSV file so that users can track their performance across multiple study sessions.

The progress chart uses matplotlib to visualize improvement over time.

A Pomodoro timer was included to encourage focused study sessions and regular breaks.

## Challenges

One challenge was handling different question types returned by the API, including multiple-choice and True/False questions.

Another challenge was storing and loading quiz data while keeping the file format consistent.
